Projects

Event Management System Platform

The Event Management System is a web and mobile platform designed to help students, faculty, and organizers manage university events easily.

Users can browse upcoming events, register, and receive notifications. Organizers can create events, track attendance, and manage registrations. The system supports secure payments for paid events and provides QR code tickets for quick check-ins. Users can also rate events and contact organizers for inquiries. It includes search and filter options, email notifications, and image uploads for event promotion.

Hikaya

Hikaya is a web application that uses OpenAI’s API to generate personalized stories for children based on their preferences.

The website allows parents to customize elements such as the child’s name, age, favorite themes, and the moral of the story.Parents can also choose cultural elements or favorite characters to make the stories feel unique. The AI then creates age appropriate, engaging, and educational stories that make stories more fun and personal.
